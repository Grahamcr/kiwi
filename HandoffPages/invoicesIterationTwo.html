Invoice Iteration Two for ACG EM

Created Dec 14 2018

Composed by Craig Graham

Key Points - Provide Invoice Iteration Two
  • This Handoff is for Design Request OB-1502: Run worker to check for audit pre-checks and Design Request OB-1468: List view for invoices
  • The goal of this next iteration is to create a List View for the Invoices that displays the status of each invoice. If the invoice has been flagged with errors, those errors can be viewed in a seperate list view.
  • Mockup can be viewed at: Invoice List View Mockup

This second iteration mockup hopes to improve the users ability to access important information related to the processing of invoices. It also redesigns the invoice details view to include addtional information related to the audit pre-check so the user can be quickly become aware of an concerns with invoice.

Style Suggestions - List View

This list view shows all of the invoices which are going through the invoice processing process, which state they are in and any concerns which have been flagged. The user can view a popover which summerizes the errors the invoice has.
