vMOX OnePortal RTMM Alerts

Created Aug 29 2018

Composed by Craig Graham

Key Points - vMOX OnePortal RTMM Alerts:
  • This is the first release of the vMOX OnePortal RTMM Alerts
  • The goal of this feature is to provide customers and prospect insight in the value RTMM Alerts provide them. The key insight we hope to show is how RTMM change user's behavior, primarly by showing that the number of RTMM alerts drop from month to month.
  • Mockup can be viewed at: vMOX OnePortal RTMM Alerts

The most difficult part of this design will be how we handle Orphan lines. In the mockup you can see how the flyout changes by clicking on the Details button for the last line in the mockup (the Details button for 231-654-8975)
