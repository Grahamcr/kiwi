vMOX Communications Management Release 3

Created Aug 22 2018

Composed by Craig Graham

Key Points - Communication Management:
  • This is the third release of the Communications Management feature
  • The goal of this release to give users the ability to do self-service sign up for each of the different reports we offer
  • Mockup can be viewed at: vMOX Communication Management Mockup Release 3

Release Three includes the new marketing landing page for the Communications Management feature, as well as all of the pages included in the self-service report sign up process. This page covers these new pages and features from a design perspective.
