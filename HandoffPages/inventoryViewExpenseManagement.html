Inventory View for ACG EM

Created Sep 20 2018

Composed by Craig Graham

Key Points - Provide Invoice Details View in OSS
  • This Handoff is for Desigh Request OB-1359: Provide Invoice Details View in OSS
  • The goal of this feature is to create a universal view for invoices in OSS. This view will breakdown the invoice and provide insights into where a user should possibly investigate further.
  • Mockup can be viewed at: ACC Portal Mockup

This is the handoff for the first iteration release of the OSS Invoice View. This view was designed to give valuable insights into any indescrepencies which might exist in the actual charges on a invoice compared to the expected charges. These insights will be displayed both graphically and textually throughout the experience.

Style Suggestions - Home Screen

The home screen starts by showing the totals and details for the charges on this invoice. Under that the screen continues by detailing the history of the invoice charges and any insights we might have that will be helpful for the user. Below that is a tabbed table that breaks down each of the cost categories for the invoice.

Style Suggestions - Line Details Flyout

The line details flyout shows the charges that add up to the totals we displayed in the tabbed table on the home view. These charges are displayed in a series of tables that can be viewed quickly by selecting the total at the top you want to see the breakdown of.
