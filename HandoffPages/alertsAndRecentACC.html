ACC Alerts Navigation Access

Created June 26 2018

Composed by Craig Graham

Key Points - ACC-606:
  • This Handoff is for Desigh Request ACC-606: Access to Alerts
  • The goal of this feature is to provide consistent access to the Alerts list throughout the application
  • Mockup can be viewed at: ACC Portal Mockup

The Alerts notification area is a new addition all of the views in ACC with the exception of the Map View which already has this functionality built into the right-rail bar. The notification dot in the header shows a count of the user's alerts.
